## ✨ 概要

**mymy** は、AI エージェントとコードプロジェクトをひとつの落ち着いたインターフェースにまとめます。ターミナル、ブラウザのタブ、設定ファイルを行き来する代わりに、ひとつのダッシュボードで：

- **エージェントを一目で** — 誰がアクティブか、アイドル中か、何を実行しているか
- **プロジェクトへ** — GitHub、GitLab、Gitea に接続されたプロジェクト
- **エージェントとチャット**、ノート作成、カレンダー管理 — すべて一箇所で
- **ロック & プライバシー** — PIN ベースのアクセス（クラウドアカウント不要、ログインフロー不要）

**一人企業**のために設計されています — あらゆる役割を兼ねる開発者ファウンダー、残りはエージェントに委ねる人。

## 🚀 機能

| | 機能 | 説明 |
|---|---------|-------------|
| 🔐 | **PIN 認証** | ローカルファーストのアクセス。デフォルト PIN `mymy`、設定で変更可能。アカウント不要、メール不要。 |
| 🤖 | **エージェント管理** | Hermes / OpenClaw エージェントをリアルタイムステータス、アバター、役割と共に表示。 |
| 📁 | **プロジェクトワークスペース** | Git リモートに接続されたプロジェクト — 進入、整理、エージェント割り当て。 |
| 💬 | **チャット** | エージェントとチャット、プロジェクトや一般トピックに紐付くセッション単位で整理。 |
| 📅 | **カレンダー** | 予定のスケジュールと管理、プロジェクトに連携。 |
| 📝 | **ノート** | Markdown ノート、PostgreSQL フルテキスト検索（FTS）、タグ、ピン留め機能。 |
| ⚙️ | **設定** | エージェントシステム（マルチインスタンス：ローカル + リモート）と Git 連携の構成。 |
| 🌐 | **i18n** | 英語、韓国語、中国語、日本語の完全サポート。 |
| 🎨 | **Linear スタイル UI** | Linear にインスピレーションを受けた集中型ダークテーマ — 一日中見ても快適。 |

## 🏗️ アーキテクチャ

| レイヤー | 技術 | 備考 |
|-------|-----------|-------|
| **フロントエンド** | Vite · React 19 · TypeScript | SPA、SSR 不要 |
| **スタイリング** | Tailwind CSS v4 | CSS 変数デザイントークン |
| **状態** | Zustand · TanStack Query · React Router | 認証 + 設定（localStorage）、サーバー状態（React Query） |
| **バックエンド** | Rust · axum | REST API、エージェント CLI に接続 |
| **データベース** | PostgreSQL 16 + pgvector | フルテキスト + セマンティック検索をひとつの DB で |
| **インフラ** | Docker Compose | 一コマンドですべて実行 |

## ⚡ クイックスタート

### 前提条件

- [Docker](https://docs.docker.com/get-docker/) & Docker Compose
- （ローカル開発用）[bun](https://bun.sh/) ≥ 1.3 および [Rust](https://www.rust-lang.org/) ツールチェーン

### Docker で実行

```bash
# 1. クローン
git clone <リポジトリ-URL> mymy && cd mymy

# 2. 環境設定
cp .env.example .env

# 3. 起動
docker compose up -d --build
```

**http://localhost:33696** を開き、PIN 入力：**`mymy`**

### ローカル開発

```bash
# フロントエンド
cd web
bun install
bun run dev      # http://localhost:5173 (HMR)

# バックエンド（別ターミナル）
cd api
cargo run        # http://localhost:33697
```

## 🔧 設定

すべての設定はアプリ内の**設定**ページ（`/settings`）にあります：

| セクション | 構成内容 |
|---------|-------------------|
| **一般** | PIN の変更 |
| **エージェントシステム** | Hermes/OpenClaw インスタンスの追加（ローカル自動検出、リモート手動） |
| **Git 連携** | GitHub、GitLab、Gitea への接続（ホスト、ポート、SSH エイリアス） |
| **情報** | バージョン & ポート情報 |

設定はブラウザの `localStorage` に保存され、Rust バックエンドと同期されます。

### ポート

mymy は一般的なサービスとの競合を避けるため **33xxx** 範囲を使用します：

| サービス | ポート |
|---------|------|
| Web（フロントエンド） | `33696` |
| API（Rust） | `33697` |
| PostgreSQL | `33432` |
