## ✨ 개요

**mymy**는 AI 에이전트, 프로젝트 파일, 운영 데이터를 하나의 로컬 인터페이스로 모아줍니다. 터미널, 브라우저 탭, 프롬프트, 흩어진 파일을 오가는 대신, 하나의 워크스페이스에서:

- **등록한 LLM 프로바이더 기반 네이티브 에이전트 생성**
- **에이전트 작업을 Drive의 `/drive/agents`, `/drive/projects`, `/drive/shared` 안에 유지**
- **에이전트 프롬프트(`AGENTS.md`, `SOUL.md`) 편집**
- **마크다운, 이미지, 미디어, PDF, docx 텍스트, 로컬 개발 서버 미리보기 열기**
- **샌드박스 러너를 통해 에이전트 명령과 장기 실행 프로세스 실행**
- **에이전트와 채팅**, 파일 첨부, 도구/검색 결과 확인, 노트/태스크/목표/재무/투자/캘린더 관리
- **잠금 & 프라이버시** — PIN 기반 접근 (클라우드 계정 없음, 로그인 플로우 없음)

**1인 기업**을 위해 설계되었습니다 — 모든 역할을 겸하는 개발자-파운더, 나머지는 에이전트에게 위임하는 사람.

## 🚀 기능

| | 기능 | 설명 |
|---|---------|-------------|
| 🔐 | **PIN 인증** | HttpOnly 쿠키 기반 서버 세션. 기본 PIN `mymy`, 설정에서 변경 가능. |
| 🤖 | **네이티브 에이전트 관리** | 에이전트 생성/삭제, 프롬프트 편집, 전역 활성 에이전트 선택, 등록된 LLM 프로바이더 기반 채팅. |
| 📁 | **Drive 워크스페이스** | `/drive/projects`, `/drive/agents`, `/drive/shared` 파일 탐색, 편집, 업로드, 삭제, 복원, 영구 삭제와 md/docx/이미지/오디오/비디오/PDF 뷰어. |
| 🧱 | **샌드박스 러너** | 파일 쓰기, 터미널, 코드 실행, 장기 실행 작업을 전용 러너와 bubblewrap/Firecracker 격리로 실행. |
| 🖥️ | **미리보기 프록시** | 에이전트가 띄운 개발 서버 포트를 토큰화된 preview URL로 열기. |
| 📁 | **프로젝트 워크스페이스** | 프로젝트별 안정적인 Drive 폴더와 채팅/작업 세션 연결. |
| 💬 | **채팅** | 마크다운/코드/검색 결과 렌더링, Drive 파일 첨부, 드래그 앤 드롭 업로드, clarify/승인 요청 인라인 처리. |
| 📊 | **투자** | 계좌, 자산, 포지션, 평가 스냅샷, 현금흐름, 관심목록, 배분 요약을 수동으로 관리. 외부 증권사 동기화나 매수/매도 실행은 없음. |
| 🧭 | **프로세스** | 샌드박스 런타임 상태, 관리 프로세스, CPU/RAM/storage, 포트, 미리보기, 로그, 중지, 강제 종료 관리. |
| 📅 | **캘린더** | 일정 예약 및 관리, 프로젝트에 연결. |
| 📝 | **노트** | 마크다운 노트, PostgreSQL 풀텍스트 검색(FTS), 태그, 고정 기능. |
| ⚙️ | **설정** | PIN, LLM 프로바이더, 에이전트 시스템, 확장, 스킬, Git 연동 구성. |
| 🌐 | **i18n** | 영어, 한국어, 중국어, 일본어 완전 지원. |
| 🎨 | **Linear 스타일 UI** | Linear에서 영감받은 집중된 다크 테마 — 하루 종일 봐도 편안한. |

## 🏗️ 아키텍처

## ⚡ 빠른 시작

### 사전 요구사항

- [Docker](https://docs.docker.com/get-docker/) & Docker Compose
- (로컬 개발용) [bun](https://bun.sh/) ≥ 1.3 및 [Rust](https://www.rust-lang.org/) 툴체인

### Docker로 실행

```bash
# 1. 클론
git clone <저장소-URL> mymy && cd mymy

# 2. 환경 설정
cp .env.example .env

# 3. 실행
docker compose up -d --build
```

**http://localhost:33696** 을 열고 PIN 입력: **`mymy`**

### 로컬 개발

```bash
# 데이터베이스
docker compose up -d db

# 프론트엔드
cd web
bun install
bun run dev      # http://localhost:5173 (HMR)

# 백엔드 (별도 터미널)
cd api
DATABASE_URL=postgres://mymy:mymy@localhost:33432/mymy cargo run
```

## 🔧 설정

모든 설정은 앱 내 **설정** 페이지(`/settings`)에 있습니다:

| 섹션 | 구성 내용 |
|---------|-------------------|
| **일반** | PIN 변경 |
| **모델 / LLM 프로바이더** | OpenAI 호환, Anthropic, Ollama, 로컬 프로바이더 엔드포인트 등록 |
| **에이전트 시스템** | 필요 시 레거시 Hermes/OpenClaw 인스턴스 관리 |
| **스킬 / 확장** | 네이티브 스킬, MCP 서버, 확장 연동 구성 |
| **Git 연동** | GitHub, GitLab, Gitea 연결 (호스트, 포트, SSH 별칭) |
| **정보** | 버전 & 포트 정보 |

설정은 브라우저 `localStorage`에 저장되며 Rust 백엔드와 동기화됩니다.

### 포트

mymy는 일반 서비스와의 충돌을 피해 **33xxx** 대역을 사용합니다:

| 서비스 | 포트 |
|---------|------|
| Web (프론트엔드) | `33696` |
| API (Rust) | `33697` |
| Sandbox runner | `33698` |
| PostgreSQL | `33432` |
